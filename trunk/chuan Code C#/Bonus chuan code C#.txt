Chúng ta sẽ lấy qui tắc đặt tên Control, Class.... theo FPT(Standard_CSharp Coding Convention.pdf) và một số tên control mình đã bổ sung ở trên để việc thiết kế và code đồng bộ. Nếu bạn nào làm không giống thì tự sửa lại dùm nha, về sau cứ thế mà làm!! Chúc mọi người làm việc vui vẻ!
